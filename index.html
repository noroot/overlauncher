Why OverLauncher?

OverLauncher is a lightweight, developer-friendly alternative to Spotlight. Built natively with Swift and SwiftUI, it offers fast application launching without the overhead of indexing and unnecessary features. Why disable Spotlight ?

Features

  • No Spotlight indexing required to just launch apps
  • Fast Search: Real-time filtering of applications as you type
  • Global Hotkey: Activate with Cmd+Space
  • Native Performance: Built with Swift and SwiftUI for optimal macOS integration
  • Application Discovery: Automatically finds apps in:
    • /Applications
    • /System/Applications
    • /System/Applications/Utilities
    • /System/Library/CoreServices
  • Keyboard Navigation: Use arrow keys to navigate, Enter to launch, Escape to close
  • App Icons: Displays application icons in search results
  • Background App: Runs silently in the background without cluttering the dock
  • No telemetry: No telemetry, no ads, pure functionality
  • Calculator: you can do simple math calculations, same as on Spotlight.

Installation

Download the latest build from the releases page:

  • Download the latest release
  • Unpack and Copy the app to /Applications.
  • Disable Spotlight indexing (optional for cleaner performance).
  • Disable Spotlight hotkey: Go to System Settings → Keyboard → Shortcuts → Spotlight and uncheck the Cmd+Space combination.
  • Run OverLauncher.
    Binary is unsigned, so you have to allow it to run through security settings in MacOS.
  • Try it out by pressing Cmd+Space.
